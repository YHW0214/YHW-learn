
可能用到

```
git remote rm origin

1.初始化本地项目
git init
2.与远程分支建立联系
git remote add origin url地址
3.同步远程数据
git fecth origin
4.切换分支
git checkout master
5.同步本地分支与远程分支
git branch --set-upstream-to=origin/master master
6.拉去远程分支，允许合并不相关的库
git pull origin master --allow-unrelated-histories
7.推送本地分支至远程
git push -u origin master
```
