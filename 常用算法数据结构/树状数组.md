1. 计算数列的前缀和、区间和
2. 时间复杂度O（log n）
3. 支持修改
4. 二进制思想

参考代码

```java
class BIT {
    int[] nums;
    int len;

    public BIT(int len) {
        this.nums = new int[len + 1];
        this.len = len;
    }

    public int query(int x) {
        int ans = 0;
        while (x != 0) {
            ans += nums[x];
            x -= lowbit(x);
        }
        return ans;
    }

    public void add(int x, int dt) {
        while (x <= len) {
            nums[x] += dt;
            x += lowbit(x);
        }
    }

    public static int lowbit(int x) {
        return x & (-x);
    }
}
```
